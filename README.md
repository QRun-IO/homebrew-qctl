# Homebrew Tap for qctl

Homebrew formulae for [qctl](https://github.com/QRun-IO/qctl) - the CLI for the QRun ecosystem.

## Installation

```bash
brew tap QRun-IO/qctl
brew install qctl
```

## Usage

```bash
# List available templates
qctl qqq list

# Scaffold a new QQQ application
qctl qqq init my-app
```
